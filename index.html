My Projects

  • Vizcap

    Built a real time image captioning application which takes images using webcam and generates caption which is then converted to speech for helping visually impaired people. Image features extracted using pre trained CNN pre trained models Resnet and VGG16,caption text generated using LSTM . Speech generated in multiple languages using Google Text to Speech API

  • Liga Footy

    Created Liga.Footy a streamlit application that uses footballl player data and generate different visuals Data from websites such as Fbref,Whoscored scraped using Selenium and Beautiful Soup Can create Radar plots,Pizza Plots, Player Season Report ,Instant Player Match Reports

  • Comment-Toxicity-Detector

    This project is used to classify comments and assign true or false values to following labels: toxic severe_toxic obscene threat insult identity_hate A model is created which predicts a probability of each type of toxicity for each comment. Data is from the Toxic Comment Classification Challenge on Kaggle.In the text preprocessing, a vectorizer is used to convert the text into vectors. It is then fed into the bidirectional LSTM model for training.
